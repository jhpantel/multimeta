Description:
*Note that each of the 27 taxa has its own directory. Within that directory are 5 files that are described in more detail below. The files model_full.txt, dry_final.txt, and JHP_JAGS.R are identical across all 27 directories. The directories differ in the contents of presence.txt and state.txt, which are unique for each taxa.

model_full.txt - A model written in JAGS, for the estimation of metapopulation parameters in the model without covariates. The file is called in the R script JHP_JAGS.R. This model is identical across all 27 directories.

dry_final.txt - A data file containing the wet / dry state of each site in each year of the survey. The data dimensions are 278x15. For each site x year, a value is either 0 (the site was not visited that year or the data is missing for that year), 1 (the site is wet), 2 (the site is dry), or a value is given between 0-1 that gives an estimate of the probability a site was dry in a given year (this estimate results from GLMM described in the manuscript section 'Data'). This data is identical across all 27 directories.

presence.txt - A data file containing the detected / not-detected state of each site in each year of the survey, both for site visits and site-revisits. The data dimensions are 278x30. For each visit occasion, a value is either 0 (the site was not visited that year or the data is missing for that year), 1 (the taxa was not detected during that occasion), or 2 (the taxa was detected during that occasion). This data is unique for each of the 27 directories (the data is thus unique for each taxa).

state.txt - A data file containing the detected / not-detected state of each site in each year of the survey. This data is compiled across site visits and site re-visits, which explains why the data is not a duplicate of the 1st 15 columns in presence.txt. If a taxa was detected either in the visit or re-visit, it is indicated as detected in this data file. The data dimensions are 278x15. For each visit site x year, a value is either 0 (the site was not visited that year or the data is missing for that year), 1 (the taxa was not detected during that occasion), or 2 (the taxa was detected during that occasion). This data is unique for each of the 27 directories (the data is thus unique for each taxa).

JHP_JAGS.R - A code file written in R that loads the data files, prepares the data for analysis using the JAGS model model_full.txt, and returns the results of the Bayesian sampling to an output file called jsample.RData. This code file is identical across all 27 directories.
